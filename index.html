MyST - 标记的结构化文本（Markedly Structured Text） ¶

MyST 是一种丰富的、可扩展的 Markdown 风味，用于技术文档和出版。

MyST 是一种 markdown 风味，它的设计是为了简单、灵活和可扩展性。这里有几个主要的特点

符合 CommonMark 标准 ✔

MyST 是 CommonMark Markdown 的一个超集。任何 CommonMark 文档也都是符合 MyST 的。

额外的创作语法 ✍

MyST 用于学术写作和技术文档的语法 扩展了 CommonMark。

可扩展的语法 🚀

MyST 提供了 角色 和 指令，使你可以扩展 MyST 的功能。

兼容 Sphinx 📄

MyST 受到 Sphinx 的启发，并带有 自己的 Sphinx 解析器。用 Markdown 编写你的 Sphinx 文档，或者从 CLI 或 使用交互式网页界面 转换现有的 RST 到 Markdown！

可以用 Python 破解 🐍

这个 MyST 解析器是建立在 markdown-it-py 包 之上的，这是一个可插拔的 Markdown Python 解析器。”

可以用 Javascript 破解 🌍

Javascript 解析器 建立在 markdown-it 的基础上，并允许你在网站中解析 MyST。
